Forward-Backward

HMMBase.forward — Function.
HMMBase.forward(a, A, L)

Compute HMMBase.forward probabilities using samples likelihoods. See Forward-backward algorithm.

source
HMMBase.forward(hmm, observations)

Example

hmm = HMM([0.9 0.1; 0.1 0.9], [Normal(0,1), Normal(10,1)])
z, y = rand(hmm, 1000)
probs, tot = HMMBase.forward(hmm, y)
